HTML

  • The body element repesents the visible content shown to the user
  • The head element contains information about the webpage
  • ul is unordered list of items, li is the things we are listing, ol is the ordered lists
  • h1 - h6 are going to be headers with h1 being the main header and should only be used once
  • section is going to define a section in a document that contains a thematic content
  • img contains information about the image that is displayed on the webpage
  • src defines the location of an image
  • The a element stands for anchor and creates a link to the same webpage or other webpages
  • The href atribute points to the URL for the link.
  • rel is the attribute that specifies the relationship between the current docutment and the linked document resource

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • By using a , we can combine 2 sections together to ass the same effects to which will in the long run shorten our code
  • text-align is the rule we can use to center a title or webpage
  • font-size will alter the font size of the text indicated

Git

  • git status: checks what branch we are currently working in
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout "branch-name": switches you to named branch
  • add -A is used to stage all files in the working branch
  • git commit -m "description of what was done" this will commit the changes done to the document and the -m is the message we add with out commit
  • git pull origin main is how you pull the base branch into your feature branch good habit to do before pushing code
  • git push oriign "branch name" is how we are going to push the new code to github
  • git branch -m "new name" is how you rename a branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in whcih a computer executes code in a script
  • A function is simply a set of instruction that tells the computer how to perform a certain task
  • an if statement is a way we add decision making into code
  • else if is how you add more than one condition
  • an array is a single varaible used to hold a group of data. typically the data is related in some way.
  • arrays are useful for storing multiple item with different data in a single variable, but they dont always have to have multiple types of data.
  • starts with an index of 0 instead of 1
  • A for loop uses the predicatable pattern of indices to perform a task on all the items in an array by allowing a single code blockl to be excuted over and over again
